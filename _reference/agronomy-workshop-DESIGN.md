# Agronomy Workshop — Anchor Reference (Refero)

> **Role for Bighorn:** the workwear/utility texture. How an apparel catalog reads as
> "tactile, jobsite-real, technical" rather than "fashion" or "tech startup."

**Source:** https://agronomywork.shop · Refero style `7fe32f5c-90ee-4149-893c-358dbb034580`
**Theme:** light · **Mood:** "monochromatic workwear utility, slightly rugged fashion-catalog"

## Why it's an anchor
Refero literally describes this one as a "strong workwear / utilitarian mood" with
"fabric textures and apparel details given prominence." It's the closest direct match
to what Bighorn sells. The lesson: monospaced technical labels + tightly-cropped real
product photography on neutral grounds = credible workwear, no gimmicks.

## Key tokens
- **Canvas:** Canvas White `#ffffff` · Soft Off-White `#eeede8` for alternating sections
- **Ink:** Rich Black `#000000` / Text Dark `#1e1e1e` · Cool Gray `#888888` muted
- **Accent:** single vivid blue `#0076ff` — ONE sharp accent only, everything else monochrome
- **Type:** semi-condensed medium-weight display (compact, commanding) + **monospace for
  small technical labels, tags, SKUs, prices** (wide 0.1em tracking — the "spec sheet" feel)
- **Radius:** `5px` default, `20px` for large blocks · **Density: compact**
- **Shadows:** none — flat UI, subtle borders + background shifts only

## Do
- Use a monospace face for SKUs, spec labels, price chips, tags — wide tracking
- Tightly-cropped real product/garment photography on neutral backdrops
- Single accent color; keep everything else black/white/gray
- Compact density, 4px spacing rhythm, sharp small radii

## Don't
- Don't add a second chromatic accent
- Don't use decorative shadows or elevation
- Don't use large rounded radii — keep components structured/precise
