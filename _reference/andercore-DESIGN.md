# Andercore — Anchor Reference (Refero)

> **Role for Bighorn:** how to do DARK correctly. Bighorn's current site is dark and
> wrong. Andercore is dark and right — because dark is carried by full-bleed real
> industrial photography with a tint, never by flat fill.

**Source:** https://andercore.com · Refero style `15fd028d-c493-47a9-8e69-0a59c6fdb14b`
**Theme:** dark · **Mood:** "rugged trade-and-logistics, digital control-panel"

## Why it's an anchor
Andercore is a trade/industrial brand. Its dark theme works because every dark section
is a **full-bleed photograph of real machinery / trucks / workers, desaturated and
tinted** — the darkness is image-driven atmosphere, not an empty navy rectangle. This is
the single most important correction for Bighorn: dark is allowed, but it must carry a
real photo. Flat dark fill = the cheap look we're killing.

## Key tokens
- **Canvas:** Midnight Void `#0b0405` near-black · Dark Shroud `#150e0f` subtle variation
- **Ink:** Pure White `#ffffff` text · Slate Text `#858182` secondary
- **Accent:** single sharp red `#e32735` (Bighorn substitutes its existing GOLD here)
- **Borders:** Ghostly Gray `#382e30` hairline
- **Type:** Archivo-style compact sans, tight `-0.02em` tracking, display 56px/40px ·
  monospace (Space Mono) for technical badges
- **Radius:** buttons `4px`, content cards `0px` (sharp)
- **Shadows:** none — hierarchy via contrast, not elevation

## Do
- Every dark section must be a full-bleed real photograph with a dark tint overlay
- One accent only, used for primary CTAs and active states
- Compact tight-tracked type; mono for technical badges
- Sharp-edged content cards, 4px radius on buttons only

## Don't
- Don't use flat dark fill as a section background with no image — this is the failure mode
- Don't add extra saturated colors beyond the single accent
- Don't use drop shadows or soft elevation
- Don't use organic/soft decorative shapes — keep it technical
